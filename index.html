CSS使元素水平垂直居中的方法总结

2016-10-23

方法一：

兼容主流浏览器，除了IE6

position加margin的方式，废话不多说，直接上代码




.wrap{
width: 200px;
height: 200px;
background-color: red;
position: relative;
}
.center{
width: 100px;
height: 100px;
position: absolute;
top: 0;
left: 0;
bottom: 0;
right: 0;
margin: auto;
background-color: green;
}
第一次是看别人这么写的，第一眼我是懵逼的，这也行？试了试果真可以，但是为什么可以呢？我本着啥都刨根问底的精神研究了一阵子也没研究出个啥来，这里有篇知乎的帖子有说，不过我能明白个大概，讲不清楚，这就很尴尬了。链接：
http://www.zhihu.com/question/35113393
方法二

兼容主流浏览器，除了IE6/7

display:table-cell;




.wrap{
width: 200px;
height: 200px;
background-color: red;
display: table-cell;
vertical-align: middle;
text-align: center;
}
.center{
width: 100px;
height: 100px;
vertical-align: middle;
display: inline-block;
background-color: green;
}
这种方法其实重点在于vertical-align这个属性。（设置或检索内联元素在行框内的垂直对其方式。）对wrap这个div做diaplay：table-cell处理使之变成行框，然后center这个div转化为行内快，梳理成长的也就居中了。
